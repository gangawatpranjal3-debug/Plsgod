# Daily Schedule To-Do List - User Guide

## Quick Start

### What is the Daily Schedule?
A premium time-based task planner that helps you organize your day with scheduled tasks that automatically track their status based on the current time.

---

## Creating Your First Task

1. **Open the Schedule Dialog**
   - Click the "Schedule" button in the Daily Schedule section
   - The dialog will open with empty fields

2. **Enter Task Details**
   - **Task Title**: Type what you need to do (e.g., "Study Math")
   - **Start Time**: Select hour, minute, and AM/PM
   - **End Time**: Select hour, minute, and AM/PM
   
   💡 **Tip**: Make sure the end time is after the start time!

3. **Save Your Task**
   - Click the "Schedule Task" button
   - You'll see a success message: "Task scheduled successfully! 🎯"
   - Your task will appear in the list

---

## Understanding Task Status

Your tasks will automatically change status as time passes:

### 🔵 **Upcoming Tasks**
- **Icon**: Gray clock ⏰
- **When**: Before the start time
- **What you can do**: 
  - Mark as complete
  - Wait for it to become active

### ⚡ **Active Tasks** (Premium)
- **Icon**: Blue pulsing play button ▶️
- **When**: Between start and end time
- **Special features**:
  - Premium badge visible
  - "Active Now" badge
  - Extension buttons available
  
- **What you can do**:
  - **Extend +10 minutes** (Blue button)
  - **Extend +30 minutes** (Purple button)
  - **Extend +1 hour** (Orange button)
  - Mark as complete
  - See how much you've extended (+Xmin badge)

### ⏹️ **Expired Tasks**
- **Icon**: Gray X ❌
- **When**: After the end time
- **Appearance**: Faded (50% opacity)
- **Badge**: "Time Expired" in red
- **What you can do**:
  - Mark as complete (if you finished it)
  - Nothing else - time's up!

---

## Extending Active Tasks

When a task is active, you can extend it as many times as you need:

1. **Click an extension button**:
   - +10 min: Quick 10-minute extension
   - +30 min: Medium 30-minute extension  
   - +1 hr: Long 1-hour extension

2. **Watch for the reaction**:
   - You'll see a fun emoji: 🚀 💪 ⚡ 🔥 ✨
   - The end time automatically updates
   - A badge shows total extension (+40min, etc.)

3. **Keep going**:
   - Extensions are cumulative
   - Extend as many times as needed
   - All extensions are tracked

## Viewing Your History

### Today's View (Default)
- Shows all tasks scheduled for today
- Updates in real-time
- Sorted by start time

### 7-Day History
1. Click the "History" button
2. See tasks from the last 7 days
3. Grouped by date with task counts
4. Scroll through past days
5. Click "Today" to go back

## Task Completion

### Marking Complete:
1. Click the "Complete" button on any task
2. Task gets a green checkmark ✓
3. Text shows strikethrough
4. Appears faded

### Undoing Completion:
1. Click the "Undo" button
2. Task returns to its normal state
3. Status (upcoming/active/expired) restored

## Common Questions

### Q: What happens at midnight?
A: Tasks are grouped by date. Today's tasks won't show in tomorrow's view automatically.

### Q: Can I edit a scheduled task?
A: Not yet - you can extend it while active or delete and recreate it.

### Q: How do I delete a task?
A: Currently, wait for it to expire and it will move to history, or mark it as complete.

### Q: Can I schedule tasks for future dates?
A: Currently only today's date is supported.

### Q: Do my tasks sync across devices?
A: Tasks are stored locally in your browser. Cloud sync coming soon!

### Q: What if I close the browser?
A: Your tasks are saved automatically and will be there when you return.

## Troubleshooting

### Task not showing up?
- Check if you scheduled it for today
- Look in the History view
- Refresh the page

### Status not updating?
- Wait a few seconds (updates every second)
- Check your system clock
- Refresh the page

### Extension buttons not working?
- Make sure task is active (between start and end time)
- Check if task is expired
- Try refreshing the page

### Can't schedule a task?
- Make sure you entered a title
- Check that end time is after start time
- Verify all fields are filled
